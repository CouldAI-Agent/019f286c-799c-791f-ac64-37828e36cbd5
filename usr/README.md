# Cybersecurity Dashboard UI

A responsive, dark-mode mock dashboard for visualizing cybersecurity scanning results, web form analytics, and vulnerability metrics. This Flutter application serves as a frontend prototype demonstrating a modern, glassmorphic layout, data tables, and analytics cards without active scanning functionality.

## Features

* **Dark Mode Theme**: Professional, neon-accented dark theme tailored for security tools.
* **Responsive Layout**: Adapts gracefully to desktop, tablet, and mobile dimensions.
* **Navigation Rail**: Quick access to Dashboard, Scanner, Results, Reports, and Settings.
* **Mock Analytics Data**: Visualizes security scores, risk levels, and form submission points.
* **Result Tables**: Structured data view for simulated scan results, HTTP statuses, and extracted fields.

## Tech Stack

* Flutter
* Dart

## Setup and Run

1. Ensure you have the Flutter SDK installed.
2. Clone this repository and navigate to the project directory.
3. Run `flutter pub get` to fetch any required dependencies.
4. Run `flutter run` to launch the application.
5. The application works across Web, macOS, Windows, Linux, iOS, and Android.
